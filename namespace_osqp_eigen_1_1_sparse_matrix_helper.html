OsqpEigen::SparseMatrixHelper namespace

SparseMatrixHelper namespace is a namespace that contains helper function to handle osqp matrix.

Contents

  • Reference
    • Functions

Use it to create to update or manage an osqp sparse matrix. osqp sparse matrix in compressed-column or triplet form.
